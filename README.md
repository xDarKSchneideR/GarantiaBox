# GarantiaBox
Gestor de garantias de productos


❓ ¿Por qué mis datos no aparecen en otro navegador?

GarantíaBox guarda todos los datos LOCALMENTE en tu dispositivo por 
privacidad. Cada navegador (Chrome, Safari, Firefox) tiene su propio 
almacenamiento separado.

✅ SOLUCIÓN 1: Instala la app en tu pantalla de inicio (así siempre 
usas el mismo "contenedor")

✅ SOLUCIÓN 2: Usa la función "Backup" regularmente y restaura en 
otros navegadores si es necesario

✅ SOLUCIÓN 3: Usa siempre el mismo navegador para acceder a la app
